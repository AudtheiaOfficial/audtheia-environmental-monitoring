Species Detection

Try the Porifera classifier

Upload any image to run it through official-porifera-classifier-ju8er/12, a custom classifier trained on 16,915 labeled images across 126 marine sponge species using RF-DETR (Medium). Detections render as bounding boxes directly on your image. No account or setup required.

Drag an image here, or click to browse JPEG · PNG · WebP · GIF · BMP · max 10 MB
Detections

    Detection runs through a shared proxy server hosted on Render.com’s free tier. The first request after a period of inactivity may take 30–50 seconds while the server starts. The demo is rate-limited to 5 detections per IP per hour. To run detections without these constraints, set up your own Roboflow workspace using the Audtheia setup guide.
