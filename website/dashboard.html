Airtable credentials
In Airtable, click your profile icon (top right) → Builder Hub → Personal access tokens. Create a token, add the data.records:read scope, and add your base under Access. Copy the value starting with pat — it is only shown once.
Open your base in Airtable. The URL reads airtable.com/{Base ID}/{Table ID}/... — copy the first segment starting with app.
Table IDs
